﻿﻿﻿# 世界背景与生活层级：世界如何把人塑造成现在这样

本文是 `virtual-character-realism-design-zh` 的世界背景设计指导。

这里说的世界背景，不是传统世界观百科，不是地图、国家、种族、魔法体系、科技名词和组织列表。它要回答的是一个更贴近角色的问题：

```text

这个世界每天怎样训练一个人？

角色处在这个世界的哪一层？

她把哪些外部规则误以为是自己的性格？

```

世界不是角色站着的背景板。世界是人格的模具。它用住房、价格、教育、礼貌、风险、羞耻、阶层、资源和日常小事，一遍又一遍地把人塑造成现在这样。

## 1. 核心原则

世界背景的目标不是让设定更大，而是让角色更有来处。

不要先问“这个世界有哪些国家”。先问：

- 普通人醒来后第一件担心的事是什么？

- 什么东西涨价了，什么东西变得危险了？

- 高层、中层、底层各自怎样避免掉下去？

- 角色从小被什么生活规则反复训练？

- 她现在的性格里，哪些其实是世界留下的伤、礼貌、警觉或欲望？

世界背景要落到生活细节里。一个角色的自尊、现实、冷淡、贪婪、体面、谨慎、讨好、嘴硬、独立，都可能不是抽象性格，而是环境长期塑造出来的生存方式。

```text

不是：这是一个阶层固化的城市。

而是：这座城市的人从小知道电梯分三种。高层专用电梯不报楼层，中层电梯要刷证，底层货梯总有一股潮味。角色小时候第一次进错电梯，被保安叫出来。她从那以后进任何门前都会先看一眼门口有没有权限灯。

```

## 2. 世界当前状态

世界当前状态不是时代背景介绍，而是生活压力的来源。

先判断这个世界正在经历什么：

- 上升期：机会还在，但竞争越来越早，所有人都相信努力还有用。

- 衰退期：旧秩序还在装作正常，普通人已经开始偷偷省东西。

- 战后期：人们学会不问过去，街上有很多缺口，但没人提。

- 重建期：新规则很多，旧习惯还没死，大家一边期待一边警惕。

- 监控化：安全压倒自由，所有人都学会说正确的话。

- 资源紧张：食物、住处、药、通行证、能源、信息或身份变成硬资源。

- 阶层固化：人能动，但很难动。上升成本高，下坠速度快。

- 信仰崩塌：大家嘴上还说价值，心里只信眼前能拿到的东西。

- 技术扩张：效率变高，人的失误空间变少，慢的人被系统丢下。

写世界状态时，要写它怎样进入日常：

```text

世界状态：资源紧张。

生活细节：超市不再缺货，但每个货架都有购买上限。人们排队时不再聊天，只盯着前面人的篮子。孩子会被大人教育“喜欢也不能拿第二份”。

角色影响：她长大后不喜欢别人碰她的抽屉。不是小气，是从小知道备用的东西会救命。

```

```text

世界状态：监控化。

生活细节：大家习惯在公共场合把玩笑说完以后补一句“当然我是开玩笑”。餐厅里没人压低声音，因为压低声音本身更可疑。

角色影响：她不擅长说真话。不是虚伪，是从小被训练成先判断这句话能不能被第三个人听见。

```

## 3. 生活层级

不要只写高层、中层、底层的身份。要写他们怎么活。

每一层都写这些问题：

- 住在哪里？窗外是什么？门口有什么人？

- 吃什么？什么东西可以浪费？什么东西必须存着？

- 生病找谁？有没有时间休息？

- 犯错以后有没有退路？

- 小孩从小被教什么？

- 爱情、婚姻、身体、忠诚和背叛在这一层意味着什么？

- 他们怎么看上层、下层和自己？

### 3.1 高层

高层不是“有钱”这么简单。高层的核心是：他们拥有缓冲。

高层犯错后有人收拾，高层表达需求时不必说得太直，高层可以把风险外包给别人。

```text

高层生活细节：

- 他们很少自己排队，也很少当场争执。

- 他们不直接说“我要”，而是说“这件事可以考虑一下”。

- 他们从小被教会控制表情，因为真正的命令不需要提高声音。

- 他们的房间通常很安静，安静本身就是资源。

```

高层角色常见内化：

- 认为别人的服务是环境的一部分，不一定意识到这是特权。

- 不习惯被拒绝，但会把不悦包装成礼貌。

- 更擅长暗示、试探和让别人自己领会。

- 对“失控”和“丢脸”格外敏感，因为体面是权力的一部分。

- 亲密关系里可能不直接索取，而是等对方读懂。

高层对亲密关系的默认共识：

- 爱情可以不被承诺绑定，但不是因为浪漫，是因为有退路。
- 婚姻是联盟，不是生存保障。感情破裂后财产谈判比情绪更重要。
- 身体是可控的，失控的身体状态（喝醉、哭泣、欲望写在脸上）比失控的情绪更丢人。
- 忠诚是体面的一部分，而不是感情的一部分——出轨的问题首先是被人知道，其次才是背叛。

防写偏：

不要把高层写成单纯傲慢。更真实的是，他们可能真诚、温和、受过良好训练，同时在很多细节上默认世界会让路。

### 3.2 中层

中层的核心不是稳定，而是害怕掉下去。

中层通常最守规矩，也最容易对规矩产生怨气。他们既羡慕高层的轻松，又害怕底层像自己的未来。

```text

中层生活细节：

- 家里有几件不能坏的大件。一旦坏了，整个月预算都会变形。

- 小孩被反复教育“不要给别人添麻烦”。

- 他们懂一些高层规则，但学得不自然。

- 请客、穿衣、礼物、住址、学校、工作单位都会被拿来比较。

- 体面不是装饰，是防止下坠的护栏。

```

中层角色常见内化：

- 很在意别人怎么看，因为评价会影响机会。

- 对钱敏感，但不愿显得自己在意钱。

- 对底层有同情，也有距离感，甚至有一点害怕。

- 容易把“懂事”“努力”“不麻烦别人”当成优点。

- 亲密关系里不敢太狼狈，因为狼狈会让她觉得自己失去体面。

中层对亲密关系的默认共识：

- 爱情是建立在稳定之上的东西——没稳定的时候谈感情像在冒险。
- 婚姻是"正常生活"的一部分，单身太久会被议论，但嫁错了也会被议论。
- 身体欲望可以存在，但不能被看见。欲望公开化比欲望本身更羞耻。
- 忠诚同时是道德要求、面子要求和生存策略——出轨不只是感情问题，还可能毁掉社交安全网。

防写偏：

不要把中层写成普通、无特征。中层的压力很细：不是今天活不下去，而是每一步都不能错。

### 3.3 底层

底层的核心不是贫穷，而是缺少退路。

底层的人不一定悲惨，也不一定粗鲁。他们只是更早知道承诺不如现货，规则经常不是保护他们的东西。

```text

底层生活细节：

- 很多人会随身带能立刻派上用场的小东西：塑料袋、零钱、旧药、备用钥匙。

- 他们不相信“之后会补给你”，因为之后常常没有。

- 说话直接不是没教养，是绕弯需要时间和安全感。

- 生病不是医学问题，先是误工问题。

- 小孩很早学会看大人脸色，判断今天能不能要东西。

```

底层角色常见内化：

- 对资源有占有本能，拿到手才算数。

- 不轻易相信长期承诺，但会记住具体帮过自己的人。

- 被温柔对待时可能先警觉，因为温柔常常有代价。

- 对浪费非常敏感。

- 亲密关系里可能更重视实际行动，而不是漂亮话。

底层对亲密关系的默认共识：

- 爱情里承诺不如现货——不是不想要承诺，是没条件等到兑现那天。
- 婚姻可以是一种互助结构，两个人的生存能力加起来比一个人强。但这不代表没有感情。
- 身体更早被劳动、疾病和短缺磨损，身体欲望有时被压成沉默，有时是少数还负担得起的快乐。
- 忠诚常常就是"不丢下你"。背叛不一定是出轨——关键时刻缺席，也是背叛。

防写偏：

不要把底层写成苦难符号。底层也有幽默、骄傲、审美、亲密、坏脾气和不讲理的时候。真实的人不是为了证明社会问题才存在。

### 3.4 夹层

很多角色最有张力的地方不在单一层级，而在夹层。

夹层包括：

- 从底层爬上来，但身体还按底层规则反应。

- 从高层掉下来，习惯还在，资源没了。

- 表面中层，家庭债务或身份风险把她拖向底层。

- 在高层工作，却不属于高层。

- 被底层养大，被中层教育，被高层使用。

夹层角色常见内化：

- 对两个世界的规则都懂一点，但在哪边都不完全自在。

- 容易过度解释自己为什么配得上某个位置。

- 对口音、衣服、礼仪、学历、住址格外敏感。

- 亲密关系里会害怕对方看见自己“不属于这里”的部分。

```text

她已经住进了干净的公寓，但还是会把购物袋折好塞进柜子。不是节俭人设，是过去教她：能装东西的东西不要扔。

```

## 4. 日常训练

世界不是通过一条大设定塑造人，而是通过重复的小事训练人。

写日常训练时，用这个句式：

```text

因为世界反复让她经历 X，她学会了 Y。后来她把 Y 当成自己的性格。

```

常见日常训练：

- 长期排队：学会忍耐，也学会抢在别人前面。

- 经常被检查：学会提前准备解释。

- 家里缺钱：学会不提需求。

- 父母工作不稳定：学会不相信计划。

- 医疗昂贵：学会把疼痛拖到不能拖。

- 学校强调排名：学会把人分成有用和没用。

- 居住空间拥挤：学会压低声音，也学会偷一点私人空间。

- 公共环境危险：学会不背对门、不摘耳机、不走暗路。

- 服务关系强烈：学会对上礼貌，对下冷淡，或者反过来极端厌恶这种结构。

例子：

```text

世界训练：她小时候每次开口要东西，家里都会沉默几秒。

内化结果：她长大后很少说“我想要”。不是无欲无求，是需求一出口，她身体里就先出现那几秒沉默。

写入角色：她在亲密、求助、争吵里都会先把需求说成玩笑。

```

```text

世界训练：他从小住在需要刷卡进出的楼里，访客要登记。

内化结果：他习惯把“被允许进入”当成关系的证明。

写入角色：别人主动给他钥匙，比说喜欢他更有用。

```

## 5. 资源、羞耻和退路

一个世界最能塑造人的地方，常常不是法律，而是资源和羞耻。

### 5.1 资源

先写这个世界真正缺什么：

- 钱。

- 食物。

- 药。

- 居住空间。

- 清洁水源。

- 安静。

- 时间。

- 身份。

- 通行权。

- 信息。

- 被认真对待的机会。

- 不被记录的空间。

缺什么，角色就会围绕什么形成敏感点。

```text

如果世界缺安静，高层的安静会变成特权。底层角色进入高层空间后，第一反应可能不是羡慕装修，而是不适应“没有杂音”。

```

### 5.2 羞耻

每个世界都有一套羞耻系统。它告诉人：什么不该被看见。

问题：

- 在这个世界里，穷是羞耻吗？

- 没有伴侣是羞耻吗？

- 身体欲望是羞耻吗？

- 失败是羞耻吗？

- 求助是羞耻吗？

- 不合群是羞耻吗？

- 不守规矩是羞耻，还是太守规矩才羞耻？

羞耻会进入角色的语言和动作。

```text

她不是不会接受礼物。她是会先算这个礼物会不会让自己变成欠债的人。她笑着说“太贵了吧”，那不是客套，是身体在找退路。

```

### 5.3 退路

阶层差异常常体现在退路上。

- 高层犯错后可以换城市、换身份、换团队。

- 中层犯错后会消耗积蓄、人脉和体面。

- 底层犯错后可能直接失去工作、住所或安全。

角色的勇敢、谨慎、犹豫，不能脱离退路来写。

```text

同样是顶撞上司，高层角色可能觉得这是表达原则，中层角色会算房贷，底层角色先想今晚还能不能回宿舍。

```

## 6. 阶层语言和身体习惯

世界会训练人的说话方式，也会训练人的身体。

### 6.1 请求

高层请求常常像安排：

```text

“你看今天下午方便处理一下吗？”

```

中层请求常常带解释：

```text

“不好意思，我知道有点麻烦，但我这边确实有个情况……”

```

底层请求可能更直接，也可能更讨好，取决于这个世界怎样回应他们：

```text

“能不能先给我？我晚上就还。”

```

不要把这些写成固定口癖。要写它们背后的生活训练。

### 6.2 拒绝

高层拒绝可以很轻，不需要解释，只陈述边界：

```text

“这个不太方便。”

```

中层拒绝经常要给理由：

```text

“我不是不想帮，是这次真的排不开。”

```

底层拒绝可能像防御，不是懒得解释，是没有安全的解释空间：

```text

“不行就是不行。”

```

### 6.3 愤怒

不同层级的人被允许愤怒的方式不同。

- 高层可以冷脸，让别人感到后果。

- 中层要克制，因为失态会被记录。

- 底层的愤怒可能被视为麻烦、粗鲁或危险。

角色越知道自己的愤怒没有用，越可能把愤怒改写成沉默、讽刺、讨好或身体症状。

### 6.4 身体习惯

身体也有阶层记忆：

- 进门先看门禁。

- 坐下前先看椅子干不干净。

- 吃饭不夹最后一块。

- 不把包放远。

- 说话时给别人留路。

- 走在街上避开摄像头，或故意走在摄像头下。

- 在高层空间里不敢把水杯放在昂贵桌面上。

这些比“她自卑”“她警觉”更有用。

## 7. 角色落位

写角色时，不要只写她属于哪个层级。要写她和这个层级的关系。

### 7.1 原生属于

她从小就在这个层级里长大，很多规则对她来说像空气。

```text

原生高层：她不是故意不说谢谢。她以为服务和天气一样，是世界本来就有的东西。

原生底层：她不是故意现实。她只是很早知道，不现实的人会拖累别人。

```

### 7.2 爬上来

爬上来的人最容易携带旧层级的身体反应。

```text

她现在能负担昂贵餐厅，但点菜时还是会下意识看价格右侧。看到对方没看，她会短暂地羡慕一下。

```

爬上来的角色常见矛盾：

- 不想回去，也不想承认自己怕回去。

- 看不起过去的自己，又心疼过去的自己。

- 对同层新人苛刻，因为他们像她不想再看见的旧影子。

- 对高层礼貌过度，因为她还在学习“不用证明”。

### 7.3 掉下去

掉下去的人常常保留旧习惯。

```text

他已经没钱了，但还是不会当场讨价还价。他把这叫体面，旁人看着像傻。

```

掉下去的角色常见矛盾：

- 仍然使用旧层级的语言，但世界不再配合。

- 对帮助敏感，因为帮助提醒她已经失去自给能力。

- 容易把现实困难理解成尊严被剥夺。

- 对留在上层的人心情很混：羡慕、怨、羞耻同时存在。曾经是同伴，现在对方的生活她插不进去。对方的好意可能被她听成施舍，对方的沉默可能被她理解成遗忘。

### 7.4 夹在中间

夹层角色最适合长期互动，因为她的规则常常互相打架。

```text

她会用高层的礼貌讲话，用底层的方式存东西，用中层的标准责备自己。

```

写夹层时，重点不是“身份复杂”，而是：

- 她在哪些场景像上层？

- 哪些场景像下层？

- 哪些东西会暴露她不是原生属于这里？

- 她最怕谁看出来？

## 8. 世界内化

世界内化是本文件最重要的部分。

角色把世界教给她的东西，当成了自己的性格。

| 她以为自己是 | 可能的世界来源 |

|--------------|----------------|

| 独立 | 从小没人稳定接住她 |

| 懂事 | 提需求会让家里更难 |

| 现实 | 天真从来没有被奖励 |

| 冷淡 | 热情曾经带来麻烦 |

| 礼貌 | 失礼会影响机会 |

| 贪心 | 资源总是不够 |

| 胆小 | 冒险很少被奖励，犯错代价太高 |

| 小气 | 备用物资救过命 |

| 不信承诺 | 承诺经常没有兑现 |

| 讨厌麻烦别人 | 求助曾经带来羞辱 |

| 控制欲强 | 不控制就会出事 |

写法：

```text

世界规则：这个城市里，求助会留下记录。

角色内化：她把“不求助”当成自尊。

外显行为：受伤后先自己处理，处理不了也说没事。

关系后果：别人越关心，她越紧张，因为关心会逼她暴露记录。

```

```text

世界规则：中层家庭的孩子不能出错。

角色内化：她把稳定当成爱的一部分。

外显行为：约会迟到五分钟会提前道歉三次。

关系后果：对方随性时，她不是觉得可爱，而是觉得危险。

```

把世界内化接到已有模块：

- 接角色底色：世界把她训练成什么质地。

- 接本性与选择：冲突时她先护住什么，因为世界曾经夺走过什么。

- 接三重面相：不同层级、不同空间会触发不同面相。

- 接晕染：一个动作里同时有欲望和阶层羞耻。

- 接关系变化：对方如果理解她的世界规则，关系会松动；如果嘲笑这些规则，关系会后退。

## 9. 世界持续施压

世界不能只在开场介绍一次。长期互动中，世界要持续回应角色选择。

持续施压不是强行制造大事件。它可以很小：

- 房租涨了。

- 通行规则改了。

- 工作名额少了。

- 旧身份快过期了。

- 某个区域开始查证。

- 药变贵了。

- 高层一句话改变底层一天。

- 中层为了保住位置开始互相监视。

- 新闻里开始使用新的词，所有人都假装懂。

- 某个节日到了，角色必须回到原生阶层。

写世界持续施压时，问：

```text

这个世界今天怎样提醒角色：你不能完全自由？

```

例子：

```text

角色刚学会向用户求助。第二天，系统通知所有求助记录会影响评级。

这不是为了虐角色，而是让世界保持自己的逻辑：在这个世界里，求助从来不是免费的。

```

```text

角色开始相信对方的承诺。与此同时，她工作的区域开始裁员。

她不是突然不信任对方，而是世界又一次提醒她：长期计划经常会被外部力量打断。

```

世界压力要有分寸。不要每次互动都砸大灾难。更真实的世界压力是低频、稳定、无法完全逃开。

## 10. 写入分流

世界背景要分流写入，不要把所有东西塞进世界书。

| 内容 | 写入位置 | 写法 |

|------|----------|------|

| 世界当前状态 | 世界书 / 预设 | 写公共环境和长期压力 |

| 生活层级 | 世界书 | 写高层、中层、底层的生活方式，不只写身份 |

| 日常训练 | 角色卡 / 记忆 | 写角色被世界训练出的具体习惯 |

| 资源与羞耻 | 世界书 + 角色卡 | 世界书写公共羞耻，角色卡写她内化了哪一条 |

| 阶层语言 | 角色扮演规则 / 世界书 | 写不同层级请求、拒绝、发怒的方式 |

| 角色落位 | 角色卡 | 写她属于、爬上、掉下或夹在哪一层 |

| 世界内化 | 角色底色 / 本性与选择 | 写她把什么世界规则误认为性格 |

| 持续施压 | 预设 / 状态规则 / 世界书触发 | 写世界怎样周期性回应角色选择 |

| 具体经历 | 记忆 | 写她第一次学会某条规则的事件 |

### 10.1 世界书

世界书适合写公共规则：

```text

这座城市的中层家庭普遍害怕下坠。体面、稳定工作、孩子成绩、居住区和医疗资格会被反复比较。人们很少直接承认恐惧，通常把恐惧包装成“为你好”“懂事”“别冒险”。

```

### 10.2 角色卡

角色卡适合写世界在她身上留下的结果：

```text

她来自害怕下坠的中层家庭。她把“不麻烦别人”当成礼貌，也当成自尊。真正想要什么时，她会先说“算了”，观察对方是否还愿意追问。

```

### 10.3 记忆

记忆适合写具体事件：

```text

小时候她因为忘带证件被挡在医院缴费窗口外。母亲没有骂她，只是沉默地站了很久。从那以后，她出门前会检查包三遍。

```

### 10.4 预设

预设适合写世界如何持续运作：

```text

在长期互动中，世界压力应以生活细节持续出现：价格、证件、排队、工作稳定性、空间权限、他人评价。不要只在开场介绍世界，也不要把世界压力写成连续大灾难。

```

### 10.5 状态规则

状态规则适合记录可变化的外部压力：

```text

资源压力：低 / 中 / 高

阶层暴露风险：低 / 中 / 高

声誉压力：低 / 中 / 高

求助成本：低 / 中 / 高

退路余量：无 / 少 / 可用

```

状态变化要能反过来影响角色：

```text

当退路余量降低时，角色更容易回到旧防御：少提需求、拒绝冒险、把亲密请求说成玩笑。

```

## 11. 快速问题

如果时间很少，先问这 12 个：

1. 这个世界现在是在上升、衰退、战后、重建、监控化、资源紧张，还是阶层固化？

2. 普通人明天最担心什么？

3. 高层最不需要亲自面对什么？

4. 中层最怕失去什么？

5. 底层最早学会不相信什么？

6. 这个世界最稀缺的资源是什么？

7. 在这个世界里，什么事情最丢脸？羞耻怎样影响人的日常选择？

8. 角色属于哪个层级？是原生属于、爬上来、掉下去，还是夹在中间？

9. 她从小被哪一种日常小事反复训练？

10. 她把哪条世界规则误以为是自己的性格？

11. 如果她做出改变，世界会用什么小事把她拉回旧习惯？

12. 这些内容分别应该写进世界书、角色卡、记忆、预设还是状态规则？

## 12. 常见错误

### 12.1 写成设定百科

不要开头就写国家、地名、组织、年表。除非这些东西会进入角色每天的生活，否则先放后面。

```text

不如先写：她每天坐的通勤车要经过三次身份检查。

```

### 12.2 只有高层，没有普通人

很多世界只写王室、组织、公司、神明和战争，却没有人怎么买菜、租房、看病、谈恋爱。没有普通生活，角色就没有被世界养大的痕迹。

### 12.3 底层只负责悲惨

底层不是苦难展览。底层也有玩笑、节日、审美、面子、攀比、爱情和坏习惯。只写惨，会把人写成议题道具。

### 12.4 阶层影响写成道德判断

不要写“高层虚伪，中层怯懦，底层粗鲁”。写生活条件怎样训练出这些表现。每个层级都有人味，也都有盲区。

### 12.5 世界不回应角色

如果世界只在开场出现，后面角色像在白房间里互动，这个世界就是无效的。世界应该通过价格、规则、空间、证件、他人眼光、工作和风险持续出现。

### 12.6 把世界压力写成连续灾难

世界压力不等于每轮都出大事。长期压力常常更小：一句通知、一张账单、一次检查、一个不能进入的门、一句“你不懂规矩”。

### 12.7 角色没有内化

世界背景最后必须落回角色：

```text

这个世界让她学会了什么？

她把什么误以为是自己的性格？

她现在如何在关系里重复这条规则？

```

如果回答不了这些问题，世界背景还没有真正进入角色。
