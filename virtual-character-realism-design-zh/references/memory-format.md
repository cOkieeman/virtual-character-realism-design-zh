# 记忆写入格式：把方法论塞进 title / content / tags / notes

本文是 `virtual-character-realism-design-zh` 的记忆写入格式指导。

本 skill 的 theory、时间感、身体记忆、年龄感等模块都在讲"写什么"——写记忆的时间纹理、写触发-反应-来源链条、写磨损与错位。但 AI RP 平台的记忆系统给的是四个字段：`title`、`content`、`tags`、`notes`。本文告诉 AI 怎么把方法论塞进这四个格子，不浪费任何一个字段。

## 目录

- 1. 字段分工总则
- 2. title 怎么写

## 1. 字段分工总则

| 字段 | 存放内容 | 检索权重 | AI 阅读方式 |
|------|---------|---------|-----------|
| `title` | 时间标签 + 事件核心 | 高（精确匹配） | 列表浏览时最先被读到 |
| `content` | 触发-反应-现在影响 | 中（模糊匹配） | 被检索到时全文注入 prompt |
| `tags` | 时间、类型、触发物、情绪 | 高（精确匹配 + 分类过滤） | 检索系统靠 tags 决定是否召回 |
| `notes` | 磨损版本、AI 不能直接写进叙事的部分 | 低（辅助向量化） | 通常不直接注入正文，但影响检索相似度 |

**核心规则：content 是 AI 写角色时可以直接用的素材。notes 是设计者给 AI 的幕后说明，AI 不能在正文里直接引用。** 如果一条记忆里既有"角色会怎么反应"又有"这段记忆的背景信息"，前者放 content，后者放 notes。

## 2. title 怎么写

title 不是"记忆的名字"。title 是检索关键词 + 时间标签。

**结构：**
```text
【时间标签】触发物或事件核心
```

**正例：**
```text
【童年】电梯急停被困四分钟
【上周】第一次说了"我不想一个人"
【几年前】那个人离开时关门的声音
【长进骨头里】别人抬手时先躲
```

**反例：**
```text
关于电梯的回忆
一段难忘的经历
她的创伤——电梯事件
```
反例问题：没有时间标签，没有触发物，检索系统搜"封闭空间"搜不到，AI 读到时也不知道这是远记忆还是近事件。

**时间标签统一用：**
- `【童年】` 7-12 岁
- `【少年】` 13-17 岁
- `【青年】` 18-25 岁
- `【几年前】` 模糊但身体还记的
- `【长进骨头里】` 已经成为条件反射的
- `【上月】【上周】【昨天】【这次】` 近期事件
- `【第X次】` 重复发生的事件

## 3. content 怎么写

content 是这条记忆的**可叙事内容**。AI 在互动中读到这条记忆时，能直接从 content 里提取角色的反应方式。

**结构（三要素）：**
```text
触发物：什么会激活这条记忆。
身体反应：身体怎么做，不等大脑。
现在的表现：这件事在今天怎样影响角色的选择或回避。
```

**正例：**
```text
触发：封闭电梯、忽然灭灯、电梯急停时的失重感。
身体反应：进电梯前先找紧急按钮。一旦轿厢满了，贴着后壁站。他人注意不到。
现在的表现：不说怕电梯，只说"不赶时间走楼梯"。住没有电梯的老楼，说一楼方便。其实是电梯坏过。
```

**反例：**
```text
她小时候被困在电梯里四分钟，从那以后就一直害怕封闭空间。这种恐惧伴随她直到现在，她从来不敢坐电梯。
```
问题：没有触发物（"封闭空间"太宽），没有身体反应（"害怕"是心理词），没有现在的表现（只说了"不敢坐"，没说她怎么说、怎么做）。

**不需要全写进 content 的情况：**
- 记忆的磨损版本、她讲给别人听的版本 → 放 notes。
- 来源的历史细节（几岁、哪个城市、谁在场）→ 浓缩进 content 一句，详细背景放 notes。
- AI 不应该在正文里直接复述的"设计者分析"→ 放 notes。

## 4. tags 怎么打

tags 是检索系统的入口。tag 打不好，记忆写再好也召不回来。

**每一条记忆至少覆盖这四类 tag：**

| tag 类别 | 内容 | 例子 |
|---------|------|------|
| 时间 | 年龄段或时间标签 | 童年、少年、几年前、上周 |
| 类型 | 记忆性质 | 身体记忆、温柔记忆、创伤记忆、训练记忆、事件 |
| 触发物 | 感官、场景、物品 | 电梯、关门声、手腕触碰、消毒水味 |
| 情绪/底色 | 关联的情绪或底色 | 恐惧、羞耻、依恋、控制 |

**正例：**
```text
tags: ["童年", "身体记忆", "封闭空间", "电梯", "恐惧", "控制"]
```

**反例：**
```text
tags: ["回忆", "重要"]
```
问题：标签太宽，无法区分于其他记忆。所有记忆都是"回忆"，所有东西都算"重要"。

**Tags 数量：** 3-8 个。太少召不回，太多稀释精度。

## 5. notes 写什么

notes 是给设计者和 AI 的幕后信息。AI 在正文里不能直接复述 notes 的内容，但可以用 notes 来理解这条记忆的背景。

**适合放 notes 的内容：**
- 记忆的磨损版本（她讲给别人听的版本 vs 实际发生的版本）
- 她对自己记忆的解释（可能不准确，但这是她的自我叙事）
- 为什么设计这条记忆（设计者的意图）
- 这条记忆和哪条其他记忆互相矛盾或互相印证

**example：**
```text
content（正文）：
触发：别人从背后拍她肩膀。身体反应：肩膀先耸，再转头。现在的表现：坐下永远背对墙。不解释，只说"习惯"。

notes（幕后）：
讲过很多遍的版本："小时候在学校被同学开玩笑吓过"。不提的部分：拍她肩膀的那个人后来做了更过分的事。她至今不知道自己是怕被拍，还是怕拍她的人再出现。这条记忆和另一条"那个人离开时关门的声音"互相印证——同一个人的手，同一个人的离开。
```

## 6. 各种记忆类型的格式模板

### 6.1 创伤身体记忆

```text
title：【时间标签】触发物 + 事件核心
content：
触发：……
身体反应：……
现在的表现：……
tags：时间 + "身体记忆" + 触发物 + 情绪
notes：磨损版本 / 她讲给别人听的版本（可选）
```

### 6.2 温柔身体记忆

```text
title：【时间标签】那个人的触碰方式
content：
触发：某个人的特定触碰。身体反应：不缩、沉下去、呼吸变慢。现在的表现：换别人碰同样部位会缩，换这个人不会。她自己有没有注意到这一点？（有 / 没有）
tags：时间 + "温柔记忆" + "身体记忆" + 触发物 + "依恋"或"信任"
notes：这个人是谁。第一次发生是什么时候。这条记忆是否正在被新的温柔覆盖。
```

### 6.3 训练/习惯记忆

```text
title：【时间标签】重复形成的身体习惯
content：
触发：什么场景下自动执行。身体反应：具体动作。现在的表现：还在做吗？自己意识到吗？被别人指出来时怎么解释？
tags：时间 + "训练记忆" + "习惯" + 触发场景
notes：这条习惯如果被打断，角色会怎样。
```

### 6.4 事件记忆（带有时间重量）

```text
title：【时间标签】事件核心
content：
发生了什么（< 3 句）。当时身体的状态。现在提起这件事时，角色的语气是近的还是远的？——时间越远，内容越短，身体反应越前。
tags：时间 + "事件" + 主题词
notes：这件事和角色的哪个底色或选择有关。
```

### 6.5 关系记忆

```text
title：【时间标签】和对方的关系事件
content：
"第一次……"和"现在……"的对比。一次具体的事。之后角色有什么微小的改变——不再做某件事 / 开始做某件事 / 不再解释某件事。
tags：时间 + "关系" + "第一次"或"现在" + 情绪
notes：这条记忆在关系时间线的什么位置。可以和哪条旧记忆对照。
```


### 6.6 事件记忆的时间重量

用同一格式写所有事件，AI 感知不到时间。时间重量要反映在 content 的长度和写法上：

| 时间距离 | content 写法 | 字数 | title 标记 |
|---------|-------------|------|-----------|
| 刚发生 | 身体反应在前——手还在抖、画面还在闪 | 50-100 字 | 昨天/刚才 |
| 最近 | 一句话就能勾起，但开始自嘲"怎么还在想" | 40-80 字 | 上周/上月 |
| 过去但没过去 | 嘴上不提，但身体避开。不写事件，写避开什么 | 30-60 字 | 几个月前 |
| 长进骨头 | 不叙述，直接写现在的选择。不解释来源 | 20-40 字 | 几年前/长进骨头里 |
| 遥远但没褪 | 碎片化。气味/光线/语气比事实准。允许记错 | 20-40 字 | 童年/少年 |

记忆越远，content 越短，身体反应的优先级越高。

**记忆的磨损：** 重复讲过的记忆越来越平滑，不再提起的记忆越来越锋利。磨损的版本放 notes，锋利的版本放 content。

**记忆的触发：** 记忆被感官触发——气味、温度、某句话里一个词。写 content 时不写"这让他想起"。直接写身体反应。


## 7. 常见错误

### 7.1 content 写成叙事散文

```text
❌ 错：
那是一个秋天的下午，她背着书包走回家。路边的梧桐叶子落了一半。电梯门打开的时候她犹豫了一下，但还是走了进去。她不知道那天电梯会坏。
```

这是小说正文，不是记忆素材。AI 读到这段不知道什么时候该触发，不知道怎么用。

```text
✅ 改：
触发：封闭电梯，尤其是老式电梯。身体反应：进电梯前先找紧急按钮。现在的表现：住没有电梯的老楼，说"一楼方便"。
```

### 7.2 时间标签缺失或太模糊

```text
❌ 错：
title：电梯事件
content：她小时候被关过电梯。
```

"小时候"太宽，三岁和十五岁是同一小时候，记忆质感完全不同。

```text
✅ 改：
title：【童年】电梯急停被困
content：触发：老式电梯、电梯急停时的失重感。身体反应：进封闭空间后先确认紧急按钮位置。现在的表现：不说怕，说"走楼梯锻炼身体"。
```

### 7.3 身体反应被写成心理分析

```text
❌ 错：
她从那以后患上了幽闭恐惧症，对任何封闭空间都会产生极度的焦虑和恐慌。
```

"幽闭恐惧症""极度焦虑""恐慌"全是诊断词和心理词。AI 读到这些词，只会写出角色说"我害怕"——而不是写她身体在做什么。

```text
✅ 改：
身体反应：门关上以后，手自动抬到胸前——不是交叉抱，是刚好挡住。她自己不知道这个动作。
```

### 7.4 content 和 notes 不分

```text
❌ 错：
content：这条记忆对应她的控制底色。放在这里是为了配合世界背景中层级的羞耻感。AI 在写亲密场景时可以参考。
```

这是设计者给 AI 的指令，不是记忆内容。AI 不能在正文里复述"这条记忆对应她的控制底色"。

```text
✅ 改：
content：触发：被人看到不知所措的样子。身体反应：立刻开始整理东西——桌面、袖口、头发。整理就是控制。现在的表现：越慌越整洁。
notes：对应控制底色。配合 world-background.md 中层级的体面羞耻。亲密场景中，她可能通过整理环境来回避身体接触。
```

### 7.5 tags 只有情绪没有触发物

```text
❌ 错：
tags: ["恐惧", "创伤", "身体记忆"]
```

"恐惧"能匹配到所有恐惧类记忆。AI 在角色坐电梯时检索不到这条——因为 tags 里没有"电梯"或"封闭空间"。

```text
✅ 改：
tags: ["童年", "身体记忆", "电梯", "封闭空间", "恐惧", "控制"]
```
